# Release Notes - Version v9.5 🚀

This release marks a major milestone in UI perfection, data consistency, and browser compatibility.

## Key Highlights

### 1. Safari Layout Excellence 🍎
- **Resolved**: Extreme vertical stretching in Safari due to flexbox/aspect-ratio bugs.
- **Implemented**: Safari-specific CSS media queries and container constraints to ensure the Booking Calendar remains perfectly sized.
- **Improved**: Removed `h-full` stretching to eliminate unwanted whitespace at the bottom of the calendar in Safari.

### 2. Global Data Standardization 📏
- **Unified Logic**: Defined "Completed" status as any 'Booked' slot that is **1 minute past** its start time.
- **Universal Synchronization**: This logic is now consistent across:
    - **Admin Bookings View**
    - **Member CRM History**
    - **Analytics Reports & PDF Exports**
- **Robustness**: Stats (Total Done, Active, etc.) now reflect reality based on time, even if the database isn't manually updated.

### 3. UI & Branding Perfection 🎨
- **Perfect Calendar Circles**: Redesigned calendar day markers using `aspect-square` and `w-full` for mathematically perfect circles on all devices.
- **Modal Logic Update**: Implemented `showCloseIcon` prop. Per branding standards, redundant 'X' icons are now hidden in modals that feature a primary "Close" or "Cancel" button.
- **Unified Button Theme**:
    - **Primary Actions**: Brand gold-pink (`#CE8E94`) with shadow and active scaling.
    - **Secondary/Close Actions**: Neutral gray (`bg-gray-100`) for a clean, professional aesthetic.
- **Confirm Context**: Standardized all error and confirmation popups to follow the new UI language.

### 4. Admin Panel Improvements ⚙️
- **Default Filter**: The "Current Slots" section now defaults to **"Today"**, providing immediate focus on the day's tasks.
- **Slot Management**: Improved visual feedback for slot statuses (Enable/Disable) with clearer badges and button styles.
